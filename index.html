How it works

From idea to live preview in minutes

Choose how you want to get started — install the Cursor skill or clone the repo manually.

Install the plugin from the marketplace — the AI handles everything for you.

01

Install the skill

Search for "design-playground-nuds" in the Cursor or Claude skill marketplace and enable the plugin.

02

Describe your idea

Say "create a prototype for a checkout flow" — the AI clones the repo, creates the branch, and scaffolds the screen.

03

Iterate visually

Preview changes instantly on localhost. Ask the AI to add screens, adjust layout, change copy, add animations.

04

Share a live preview

The AI opens a PR for you — CI generates a staging-backoffice URL that anyone at Nu can access.
